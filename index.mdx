## Overview

SYNQ Bot is a powerful Discord bot with **114 commands** across **25 categories**. Use this documentation to set up features, manage your server, and get the most out of your community.

<Info>
  Use the **prefix** command to change your server's command prefix. The default prefix is typically `!` or `/` for slash commands.
</Info>

## Premium tiers

SYNQ Bot offers different premium tiers that unlock additional features:

- **Elite** (Tier 1) — Full access to core features like giveaways, tracking, moderation, and more
- **Pro** (Tier 2) — Advanced features including autoresponder, custom roles, moderation suite, and automation

Use the `premium` command to check your server's premium status. See the [Premium](/premium) page for details.
