## Premium tiers overview

| Tier | Access level | Key features |
|------|--------------|--------------|
| **None** | Free | Basic commands (help, ping, premium, report) |
| **Elite** | Tier 1 | Giveaway, Message/Voice/Invite tracking, Leaderboard, Boostrole, Greet, Autorole, Embed, Reminder, Timer, Utility |
| **Pro** | Tier 2 | Everything in Elite + Autoresponder, Customise, Customrole, Guildtag, Vanityrole, Moderation, Join-to-Create, Sticky, Ticket, Welcome |
| **Developer** | Owner only | Backup, blacklist, eval, sync, subscription management, and other owner commands |

## Checking your status

Use the `premium` command to see your server's current premium tier:

```
premium
```

Aliases: `prem`

This command is available on all servers and does not require premium.
