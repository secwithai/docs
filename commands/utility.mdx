## User info

| Command | Syntax | Example |
|---------|--------|---------|
| avatar | `avatar [@user\|userId]` | Display user avatar |
| banner | `banner [@user\|userId]` | Display user banner |
| userinfo | `userinfo [@user\|userId]` | User information |

**Aliases:** `av`, `photo` for avatar; `bn`, `userbanner` for banner; `ui` for userinfo
