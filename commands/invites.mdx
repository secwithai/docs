---
title: Invites
description: Invite tracking stats and leaderboard. For invite configuration, see Invite & Welcome.
---

Invite tracking commands display how many users each member has invited to the server. These stats are used for the invites leaderboard and ranking.

## Related commands

- **invites** — View invite stats for a user (see [Invite & Welcome](/commands/invite-welcome))
- **invited** — List members invited by a user
- **inviteinfo** — Show active invite code
- **inviter** — Show who invited a user

## Leaderboard

Use the [Leaderboard](/commands/leaderboard) command to view the invites leaderboard:

```
leaderboard invites
```

Aliases: `leaderboard i`
