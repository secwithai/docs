## ping

**Syntax:** `ping`  
**Premium:** Elite

## prefix

Set the command prefix for your server.

**Syntax:** `prefix <new prefix>`  
**Aliases:** `setprefix`  
**Premium:** Elite

**Example:** `prefix !`

---

## premium

Check the premium status of this server.

**Syntax:** `premium`  
**Aliases:** `prem`  
**Premium:** None (available on all servers)

---

## profile

Displays badges for members.

**Syntax:** `profile`  
**Aliases:** `badge`, `badges`, `pr`  
**Premium:** Elite

---

## report

**Syntax:** `report`  
**Premium:** None
