## customise

Customize server avatar, banner, and bio.

**Syntax:** `customise <subcommand> [arguments]`  
**Aliases:** `customize`, `custom`  
**Premium:** Pro

### Subcommands

| Subcommand | Syntax | Example |
|------------|--------|---------|
| avatar | `customise avatar <image_url>` | `customise avatar https://example.com/avatar.png` |
| banner | `customise banner <image_url>` | `customise banner https://example.com/banner.png` |
| bio | `customise bio {description}` | `customise bio Welcome to our server!` |
| reset | `customise reset` | `customise reset` |

### Notes

- You need the appropriate Discord permissions to change server settings.
- Image URLs must be direct links to the image file.
