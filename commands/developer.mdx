## Subscription management

### subscription

Manage premium users and servers.

**Syntax:** `subscription <subcommand> [arguments]`  
**Aliases:** `+`  
**Premium:** Developer

| Subcommand | Syntax | Example |
|------------|--------|---------|
| add | `subscription add <@user\|userId> <days> <tier>` | `subscription add @user 30 elite` |
| remove | `subscription remove <@user\|userId>` | Remove premium |
| activate | `subscription activate <@user\|userId> [planId] <serverId>` | Activate for server |
| deactivate | `subscription deactivate <@user\|userId> <planId>` | Deactivate |
| transfer | `subscription transfer <@user\|userId> <planId> <targetServerId>` | Transfer between servers |

**Aliases:** `+`, `a` for add; `-`, `r` for remove; `active` for activate; `deactive` for deactivate; `t` for transfer
