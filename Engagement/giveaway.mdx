## Starting a Giveaway

You can start a giveaway with the `gstart` command.

<CodeGroup>
```javascript Syntax
,gstart <duration> <winners> <prize> [channel]
```
```javascript Example
,gstart 1h 2 Nitro Boost
,gstart 24h 1 Nitro Boost #giveaways
```
</CodeGroup>

## Editing a Giveaway

You can edit an active giveaway with the `gedit` command.

<AccordionGroup>
    <Accordion title="Changing the host">
        Use the `gedit host` command to change the giveaway host.

        <CodeGroup>
```javascript Syntax
        ,gedit host <message_id> <@user|userId>
```
```javascript Example
        ,gedit host 1461440639311806724 @newhost
```
        </CodeGroup>
    </Accordion>

    <Accordion title="Changing the prize">
        Use the `gedit prize` command to change the giveaway prize.

        <CodeGroup>
```javascript Syntax
        ,gedit prize <message_id> <prize>
```
```javascript Example
        ,gedit prize 1461440639311806724 Nitro
```
        </CodeGroup>
    </Accordion>

    <Accordion title="Changing the duration">
        Use the `gedit duration` command to change the duration of the giveaway.

        <CodeGroup>
```javascript Syntax
        ,gedit duration <message_id> <time>
```
```javascript Example
        ,gedit duration 1461440639311806724 7d
```
        </CodeGroup>
    </Accordion>

    <Accordion title="Changing the thumbnail">
        Use the `gedit thumbnail` command to change the giveaway thumbnail.

        <CodeGroup>
```javascript Syntax
        ,gedit thumbnail <message_id> <url>
```
```javascript Example
        ,gedit thumbnail 1461440639311806724 https://example.com/image.png
```
        </CodeGroup>
    </Accordion>

    <Accordion title="Changing the image">
        Use the `gedit image` command to change the giveaway image.

        <CodeGroup>
```javascript Syntax
        .gedit image <message_id> <url>
```
```javascript Example
        .gedit image 1461440639311806724 https://example.com/banner.png
```
        </CodeGroup>
    </Accordion>

    <Accordion title="Changing the color">
        Use the `gedit color` command to change the embed color of the giveaway.

        <CodeGroup>
```javascript Syntax
        ,gedit color <message_id> <hex code>
```
```javascript Example
        ,gedit color 1461440639311806724 #ffffff
```
        </CodeGroup>
    </Accordion>
</AccordionGroup>

## Default Giveaway Setup

You can configure default settings for all giveaways with the `gsetup` command.

<AccordionGroup>
    <Accordion title="Setting a default thumbnail">
        Use the `gsetup thumbnail` command to set the default thumbnail for all giveaways.

        <CodeGroup>
```javascript Syntax
        ,gsetup thumbnail <url or attachment>
```
```javascript Example
        ,gsetup thumbnail https://example.com/image.png
```
        </CodeGroup>
    </Accordion>

    <Accordion title="Setting a default image">
        Use the `gsetup image` command to set the default main image for all giveaways.

        <CodeGroup>
```javascript Syntax
        ,gsetup image <url or attachment>
```
```javascript Example
        ,gsetup image https://example.com/giveaway.png
```
        </CodeGroup>
    </Accordion>

    <Accordion title="Setting a default color">
        Use the `gsetup color` command to set the default embed color for all giveaways.

        <CodeGroup>
```javascript Syntax
        ,gsetup color <hex code>
```
```javascript Example
        ,gsetup color #FF5733
```
        </CodeGroup>
    </Accordion>

    <Accordion title="Setting a default emoji">
        Use the `gsetup emoji` command to set the default emoji for all giveaways.

        <CodeGroup>
```javascript Syntax
        ,gsetup emoji <emoji>
```
```javascript Example
        ,gsetup emoji ðŸŽ‰
```
        </CodeGroup>
    </Accordion>

    <Accordion title="Viewing current setup">
        Use the `gsetup view` command to view all current default giveaway settings.

        <CodeGroup>
```javascript Syntax
        ,gsetup view
```
```javascript Example
        ,gsetup view
```

    <Accordion title="Resetting setup to default">

        <CodeGroup>
```javascript Syntax
        ,gsetup reset
```
```javascript Example
        ,gsetup reset
```
        </CodeGroup>
    </Accordion>
</AccordionGroup>

## Giveaway Manager Role

You can manage who has permission to run giveaways using the `gmanager` command.

<AccordionGroup>
    <Accordion title="Setting the manager role">
        Use the `gmanager set` command to assign a giveaway manager role.

        <CodeGroup>
```javascript Syntax
        ,gmanager set <@role|roleId>
```
```javascript Example
        ,gmanager set @giveawayManagers
```
        </CodeGroup>
    </Accordion>

    <Accordion title="Viewing the current configuration">
        Use the `gmanager config` command to view the current giveaway manager role configuration.

        <CodeGroup>
```javascript Syntax
        ,gmanager config
```
```javascript Example
        ,gmanager config
```
        </CodeGroup>
    </Accordion>

    <Accordion title="Resetting the manager role">
        Use the `gmanager reset` command to remove the configured giveaway manager role.

        <CodeGroup>
```javascript Syntax
        ,gmanager reset
```
```javascript Example
        ,gmanager reset
```
        </CodeGroup>
    </Accordion>
</AccordionGroup>

## Banning Users or Roles

You can ban users or roles from participating in giveaways with the `gban` command.

<AccordionGroup>
    <Accordion title="Banning a user or role">
        Use the `gban add` command to ban a user or role from all giveaways.

        <CodeGroup>
```javascript Syntax
        ,gban add <@user|userId / @role|roleId>
```
```javascript Example
        ,gban add @user
        ,gban add @giveawayban
```
        </CodeGroup>
    </Accordion>

    <Accordion title="Removing a ban">
        Use the `gban remove` command to remove a ban from a user or role.

        <CodeGroup>
```javascript Syntax
        ,gban remove <@user|userId / @role|roleId>
```
```javascript Example
        ,gban remove @user
        ,gban remove @giveawayban
```
        </CodeGroup>
    </Accordion>

    <Accordion title="Viewing the ban list">
        Use the `gban list` command to view all users and roles currently banned from giveaways.

        <CodeGroup>
```javascript Syntax
        ,gban list
```
```javascript Example
        ,gban list
```
        </CodeGroup>
    </Accordion>
</AccordionGroup>
