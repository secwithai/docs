## What is Boostrole?

Boostrole allows Nitro boosters to create their own custom role in the server with unique colors, icons, and names. These roles can also be shared with other members and managed with advanced configuration options.

### Enabling your boost role

Before creating your role, you must enable boostrole.

<CodeGroup>
```javascript Syntax 
boostrole on
```
</CodeGroup>

### Setting the base role

In order for the booster roles to work, you must set a base role. All booster roles will be placed under this role.

<Tip>
The base role should be a role which is above any roles with a color, otherwise the color won't be visible due to Discord's role hierarchy.
</Tip>

<CodeGroup> 
```javascript Syntax 
boostrole baserole (@role|roleId) 
```
</CodeGroup>

### Creating a boost role

Create a role with custom colors, an optional emoji icon, and a name.

<Info>You can use one or two colors to create gradient effects.</Info>

<CodeGroup> 
```javascript Syntax 
boostrole create (#color1) [#color2] [emoji] (name) 
```
</CodeGroup>

## Customizing your booster role

### Renaming your boost role

<CodeGroup> 
```javascript Syntax 
boostrole rename (newName) 
```
</CodeGroup>

### Changing the role color

<CodeGroup> 
```javascript Syntax 
boostrole color (#color) 
```
</CodeGroup>

### Changing the role icon
<CodeGroup> 
```javascript Syntax 
boostrole icon (emoji) 
```
</CodeGroup>

### Sharing your boost role
Share with a user
<CodeGroup> 
```javascript Syntax 
boostrole share (@user|userId) 
```
</CodeGroup>

### Remove shared access
<CodeGroup>
```javascript Syntax 
boostrole unshare (@user|userId) 
```
</CodeGroup>

### Remove shared access
<CodeGroup> 
```javascript Syntax 
boostrole unshare (@user|userId) 
```
</CodeGroup>

## Server configuration (Admin)

Filter role names
<CodeGroup> 
```javascript Syntax 
boostrole filter add (word) 
```
</CodeGroup>

### Set boost message
<CodeGroup> 
```javascript Syntax 
boostrole message (content) 
```
</CodeGroup>

### Set boost channel
<CodeGroup> 
```javascript Syntax 
boostrole setchannel (#channel|channelId) 
```
</CodeGroup>

### Preview boost message
<CodeGroup> 
```javascript Syntax 
boostrole testmessage 
```
</CodeGroup>

### Viewing boost roles
<CodeGroup> 
```javascript Syntax 
boostrole list 
```
</CodeGroup>

### View boostrole configuration
<CodeGroup> 
```javascript Syntax 
boostrole config 
```
</CodeGroup>
