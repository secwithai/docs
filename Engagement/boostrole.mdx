## What is Boostrole?

Boostrole allows Nitro boosters to create their own custom role in the server with unique colors, icons, and names. These roles can also be shared with other members and managed with advanced configuration options.

## Enabling your boost role

Before creating your role, you must enable boostrole.

<CodeGroup>
```javascript Syntax 
boostrole on
```
</CodeGroup>

## Disabling your boost role

You can temporarily disable your boost role without deleting it.

<CodeGroup>
```javascript Syntax 
boostrole off 
```
</CodeGroup>

## Creating a boost role

Create a role with custom colors, an optional emoji icon, and a name.

<Info>You can use one or two colors to create gradient effects.</Info>

<CodeGroup> 
```javascript Syntax 
boostrole create (#color1) [#color2] [emoji] (name) 
```
</CodeGroup>

## Renaming your boost role

<CodeGroup> 
```javascript Syntax 
boostrole rename (newName) 
```
</CodeGroup>

## Changing the role color

<CodeGroup> 
```javascript Syntax 
boostrole color (#color) 
```
</CodeGroup>

## Changing the role icon
<CodeGroup> 
```javascript Syntax 
boostrole icon (emoji) 
```
</CodeGroup>

## Sharing your boost role
Share with a user
<CodeGroup> 
```javascript Syntax 
boostrole share (@user|userId) 
```
</CodeGroup>

## Remove shared access
<CodeGroup>
```javascript Syntax 
boostrole unshare (@user|userId) 
```
</CodeGroup>

## Remove shared access
<CodeGroup> 
```javascript Syntax 
boostrole unshare (@user|userId) 
```
</CodeGroup>

## Server configuration (Admin)
Set base role
<CodeGroup> 
```javascript Syntax 
boostrole baserole (@role|roleId) 
```
</CodeGroup>

Filter role names
<CodeGroup> 
```javascript Syntax 
boostrole filter add (word) 
```
</CodeGroup>
